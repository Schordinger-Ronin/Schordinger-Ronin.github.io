如何快捷方便的建立个人主页

回顾建立主页这三天的心路历程，给新手小白一些建议（大佬勿入）

2025-08-19

前言

在大一（亦或是大二，原谅我这个老年人）的时候我建立过一个个人主页：已经废弃的个人主页（感谢20-少计-lly），当时想的是：单纯在网络上记录自己的生活及成就（发现完全没有成就）。后来因为各种事情（大二到大四结束这段时间特别忙，也不知道忙了什么）一直没有更新。

直到大四结束的暑假，我才又有时间重新打理这些事情。由于一些变故，本人准备在QQ空间里不再更新自己的说说，并希望转用互联网进行更新发布。询问ゆき（再次感谢20-少计-lly）后其建议我使用Hugo进行搭建，以前的个人主页挂靠在中国科大官方网站上，这次的主页直接挂靠在Github上进行使用和更新（吐槽一下中国科大的邮箱简直难用至极，且各种东西依然在更新之中，比如睿客网会被全面替代），私觉得放在GitHub上让人比较放心。

下面的内容主要介绍如何利用Hugo和Github联动，方便快捷的建立一个个人主页，并指出我在建立过程中出现的bug，为后人指明一条道路。

参考如下Bilibili中的视频进行配置即可（视频中使用的是Windows电脑进行配置，本人使用的是MacOS进行配置）：

github常规部署

Git的下载和安装按照视频中展示的方法即可，网址如下：

Git官方网站

MacOS端建议先下载一个Xcode，一般会包含一个git包，如果没有那就从Homebrew下载：

1

zijianzhang@192 ~ % brew install git

在Github新建仓库的时候的Repo name一定要设置成Owner.github.io的格式(不然生成的网页名就会有后缀，不美观)，比如你的github名字叫做ronin，那你的仓库名就设置为ronin.github.io。

在把代码提交到Github仓库的时候会要求输入Github账户和密码（这里的密码输入登录密码不行，以前好像是输入登录密码，我不知道啥时候更改了），在视频的第四部分github自动部署中有设置TOKEN的教学，必须提前设置TOKEN才能把本地代码上传到Github上。（本萌新卡在这里好久）

这里的hugo.yaml文件中需要更改baseurl选项，直接复制你的生成的网页即可。你可能会问我你怎么先知道生成的网址叫什么，如果使用你的name.github.io作为仓库名，生成的网址就是https://schordinger-ronin.github.io。或者你先把文件上传到GitHub上，观察一下生成的静态网页网址是啥，然后再上传一遍就可以了。

比如你在github上生成的网页是: https://schordinger-ronin.github.io，就要在hugo.yaml文件中修改第一行的baseurl选项。直接在网址后面加入一个/符号即可(不要把这个符号忘记了，本人忘记了在这一步卡了4个小时呜呜呜)：

1

baseurl: https://schordinger-ronin.github.io/

name: deploy

# 代码提交到main分支时触发github action
on:
  push:
    branches:
      - main

jobs:
  deploy:
    runs-on: ubuntu-latest
    steps:
        - name: Checkout
          uses: actions/checkout@v4
          with:
              fetch-depth: 0

        - name: Setup Hugo
          uses: peaceiris/actions-hugo@v3
          with:
              hugo-version: "latest"
              extended: true

        - name: Build Web
          run: hugo -D

        - name: Deploy Web
          uses: peaceiris/actions-gh-pages@v4
          with:
              PERSONAL_TOKEN: ${{ secrets.你的token变量名 }}
              EXTERNAL_REPOSITORY: 你的github名/你的仓库名
              PUBLISH_BRANCH: main
              PUBLISH_DIR: ./public
              commit_message: auto deploy

自动部署的命令行代码：在dev文件夹中执行命令
git add .
git commit -m “update”
git push
